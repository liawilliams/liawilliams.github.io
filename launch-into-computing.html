    Launch into Computing

    The Launch into Computing module introduces core computing concepts for beginners: basic programming, problem solving, computational thinking, and an introduction to common tools and workflows.

    Topics & Learning Outcomes

    • Fundamentals of programming and algorithms
    • Computational thinking and problem decomposition
    • Introductory use of development tools (editors, terminals, version control)
    • Basic data structures and control flow
