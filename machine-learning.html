    Machine Learning

    The Machine Learning module introduces the underlying theories involved in machine learning and tools for exploring datasets, developing subsequent models and their evaluation.

    Topics & Learning Outcomes

    • Learn about key algortihms in machine learning
    • Understand data analytics based on machine learning using Python
    • Experience how these can be used in real-world applications
    • Gather information from various sources to analyse complex problems using machine learning tools
