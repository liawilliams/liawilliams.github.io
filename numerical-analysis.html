    Numerical Analysis

    The Numerical Analysis module introduces key aspects of mathematics and statistics for data science, highlighting the need to be aware of, and be able to critically evaluate, the range of mathematical and statistical methodology seen in data science, and to build a foundation for analysing and solving problems. R was also introduced in the module.

    Topics & Learning Outcomes

    • Root-finding algorithms (bisection, Newton-Raphson)
    • Interpolation and approximation methods
    • Numerical integration and differentiation
    • Numerical linear algebra and error analysis

    Unit 1: Types and Sources of Data and an Introduction to R

    Introduction to floating point arithmetic, rounding errors and stability of algorithms.

    Unit 2: Data Structures in R

    This unit covered the different data structures, as well as tutorials on how to perform basic algorithms, explain data frames, and use various operations on those data frames in R.
