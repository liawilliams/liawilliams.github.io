    Visualising Data

    This module introduces techniques and tools for presenting data visually. Topics include chart selection, design principles, interactive visualisation and using libraries such as D3.js and charting tools.

    Topics & Learning Outcomes

    • Principles of effective visual communication
    • Choosing the right chart and visual encoding
    • Introduction to interactive visualisations
    • Using common libraries and tools for visualization
